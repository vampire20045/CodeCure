# CodeCure
An agentic AI system for debugging code and review

CodeCure is an intelligent debugging assistant that helps developers identify, analyze, and fix issues in their code through automated analysis and review.

## Features
- Automated code analysis and bug detection
- Intelligent debugging suggestions
- Code review and quality assessment
- Support for multiple programming languages
- CLI interface for easy integration

## Quick Start
```bash
python codecure/main.py --analyze <file_path>
```

## Installation
```bash
pip install -r requirements.txt
```
